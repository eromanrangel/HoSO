This folder contains matlab code for computing HoSO descriptors as explained in:

Edgar Roman-Rangel, Diego Jimenez-Badillo, and Stephane Marchand-Maillet.
"Classification of Archaeological Potsherds using Histograms of Spherical Orientations".
ACM Journal on Computing and Cultural Heritage (JOCCH). 9(3):Online first. September, 2016.

This code works with off format files or with a set of vertices previously defined. It also contains an example 3D surface in the file tryOnMe.off.

Start by runing example.m as example.
